O(n)的算法 遍历一遍 维持当前的最大子序列和和最大后缀和 如果前n-1个的最大后缀加第n个后小于零则将最大后缀置零
```
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int maxPost = 0, maxSub = INT_MIN;
        for(int i = 0; i < nums.size(); i++){
            maxPost += nums[i];
            maxSub = maxPost > maxSub? maxPost:maxSub;
            if(maxPost < 0)
                maxPost = 0;
        }
        return maxSub;
    }
};
```
