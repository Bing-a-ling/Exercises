遍历数组，记录截至当前元素为止出现的互异元素的个数length，每出现一个新元素将该数值加一，并用其覆盖位置length-1处的元素。
遍历结束后将最终长度以后的元素统统删去。
要注意数组为空的情况
--
1.0 2019/1/1
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        if(!nums.size())
            return 0;
        int length = 1;
        int tmp = nums[0];
        for(int i = 1; i < nums.size(); i ++){
            if(nums[i] != tmp){
                tmp = nums[i];
                nums[length] = tmp;
                length++;
            }
        }
        nums.erase(nums.begin() + length, nums.end());
        return length;
    }
};
